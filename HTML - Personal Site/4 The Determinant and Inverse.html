The Determinant

Every square matrix has a quality associated with it, called the determinant. It can be used to solve sets of equations. The determinant of a matrix can be denoted in three ways - $det(A)$, $det A$, or $\begin{vmatrix}A\end{vmatrix}$.


Calculating the Determinant of a 2 x 2 Matrix


$det (A) \begin{bmatrix}a&b\\c&d\end{bmatrix} =\begin{vmatrix}a & b\\c & d\end{vmatrix} = ad-bc$


As an example:    $det (A) \begin{bmatrix}2&4\\3&5\end{bmatrix} =\begin{vmatrix}2 & 4\\3 & 5\end{vmatrix} = 10-12 = -2$



Calculating the Determinant of a 3 x 3 Matrix


We can write a 3 x 3 matrix as so:   $\begin{bmatrix}a&b&c\\d&e&f\\g&h&i\end{bmatrix}$

  1. Multiply a by the determinant of the 2 x 2 matrix that is not in the same row or column as a
  2. Do the same for -b
  3. Do the same for c
  4. Add them all together

This could also be written as so: $$det (A) \begin{bmatrix}a&b&c\\d&e&f\\g&h&i\end{bmatrix} =\begin{vmatrix}a&b&c\\d&e&f\\g&h&i\end{vmatrix} = a\begin{vmatrix}e&f\\h&i\end{vmatrix} -b\begin{vmatrix}d&f\\g&i\end{vmatrix}+c\begin{vmatrix}d&e\\g&h\end{vmatrix}  =   aei+bfg+cdh-ceg-bdi-afh$$


As an example:   $det (A) \begin{bmatrix}1&2&3\\4&5&6\\7&8&9\end{bmatrix} =\begin{vmatrix}1&2&3\\4&5&6\\7&8&9\end{vmatrix} = 1\begin{vmatrix}5&6\\8&9\end{vmatrix}-2\begin{vmatrix}4&6\\7&9\end{vmatrix}+3\begin{vmatrix}4&5\\7&8\end{vmatrix} 
=   1\cdot$$(5$ x $9) + 2\cdot(6$ x $7) + 3\cdot(4$ x $8) - 3\cdot(5$ x $7) - 2\cdot(4$ x $9) - 1\cdot(6$ x $8) = 0$



The Inverse of a Matrix

The inverse of a square matrix $A$ is usually denoted as $A^{-1}$. It is the matrix by which the original matrix $A$ should be multiplied to get the identity matrix i.e. $AA^{-1} = A^{-1}A = I$



Calculating the Inverse of a 2 x 2 Matrix

$\begin{bmatrix}a&b\\c&d\end{bmatrix}^{-1} =\dfrac{\begin{bmatrix}d&-b\\-c&a\end{bmatrix}} {\begin{vmatrix}a&b\\c&d\end{vmatrix}} = \dfrac1{ad-bc}{\begin{bmatrix}d&-b\\-c&a\end{bmatrix}}$



Calculating the Inverse of a 3 x 3 Matrix

  1. Find the determinant of the matrix. If the determinant is 0, there is no inverse matrix: $det (A) \begin{bmatrix}a&b&c\\d&e&f\\g&h&i\end{bmatrix} =   aei+bfg+cdh-ceg-bdi-afh$
  2. Transpose the original matrix: $A ^T = \begin{bmatrix}a&d&g\\b&e&h\\c&f&i\end{bmatrix}$

  3. Replace each element with the determinant of its 2 x 2 "minor" matrix. Disregard the row and column of the element you are focusing on, and calculate the determinant of the remaining values.

    i.e. For $a$, the minor matrix will be the 2 x 2 matrix including $e, f, h$ and $i$  =  $\begin{bmatrix} \huge{a} & \small{b}& \small{c}\\\small{d}& \huge{e}& \huge{f}\\\small{g}& \huge{h}& \huge{i}\end{bmatrix}$

    In this example, the value $a$ would be replaced by the value of $ei-fh$

  4. Apply the matrix of co-factors to this new matrix. This is done by reversing the sign of alternate elements, with the first element keeping its original sign.

    This pattern looks similar to a chessboard. As a visual example:   $\begin{bmatrix} +& - & +\\ - & +& - \\ +& - & +\end{bmatrix}$

    5. What you have now is known as the Adjugate Matrix.

  5. Divide each term of the adjugate matrix by the determinant  = $A^{-1}   =  \dfrac1{det (A)} \begin{bmatrix}Adj (A)\end{bmatrix}$
