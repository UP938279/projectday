How to Multiply by a Scalar

A matrix can be multiplied by a scalar, $\lambda$ (real number). You do this by multiplying every element by $\lambda$. The final matrix has the same dimensions as the original matrix.

i.e. $ A= \begin{bmatrix} a& b\\ c & d\end{bmatrix}$ $\lambda A =$ $\begin{bmatrix} \lambda\cdot a& \lambda\cdot b\\ \lambda\cdot c & \lambda\cdot d\end{bmatrix}$

$nA^T = (nA)^T$

How to Multiply with Another Matrix

For the matrix $AB$ to exist, the number of columns ($n$) in Matrix $A$ must be equal to the number of rows ($m$) in Matrix $B$. This will give a matrix with dimensions $m$ x $l$.



$A$ x $B$ = $AB$
$\uparrow$ $\uparrow$ $\uparrow$
$m$ x $n$ $n$ x $l$ $m$ x $l$


Calculate the dot product of the first row of the first matrix by the corresponding member of the first column of the second matrix i.e. $a{_1}{_,}{_1}$ x $b{_1}{_,}{_1}$ + $a{_1}{_,}{_2}$ x $b{_2}{_,}{_1}$ (for a 2-rowed matrix $a$ and a 2-columned matrix $b$).
This will give you the element of the first row and first column of your new matrix; element $a{_1}{_,}{_1}$.
You then do this for the first row with the second column, this will give you the element for the first row, second column; $a{_1}{_,}{_2}$.
You continue doing this until all of the columns are complete. Then you move to the second row - this will give you the second row for your resultant matrix.



An example

Let's do an example with numbers, to try and make things clearer.
Let Matrix $A =    \begin{bmatrix} 3 & 1 & 4\\5 & 2 & -2\end{bmatrix}$

Let Matrix $B =    \begin{bmatrix} 1 & -1\\2 & 0\\-6 & 4\end{bmatrix}$

As the number of rows for matrix $A$ is equal to the number of columns of matrix $B$, we can multiply these two matrices!
Let's examine how each box is calculated.

Matrix $AB = \begin{bmatrix} (3\cdot1 + 1\cdot2 + 4\cdot(-6)) & (3\cdot(-1) + 1\cdot0 + 4\cdot4) \\(5\cdot1+ 2\cdot2 +(-2)\cdot(-6)) & (5\cdot(-1)+ 2\cdot0 +(-2)\cdot4)\end{bmatrix}$

Matrix $AB = \begin{bmatrix} -19 & 13\\21 & -13\end{bmatrix}$

A x B ≠ B x A

It is vital to note that multiplying A x B does not equal B x A. Because of the way that it is calculated, the resultant matrix will likely be different. Depending upon what numbers are in each matrix, it may be possible to get the same resultant matrix - but it relies on luck. Ensure when multiplying matrices that you multiply them in the way that you want - the commutative property of multiplication does not apply to matrices.
